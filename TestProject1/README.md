Тестовое задание 1
Задача:
Написать веб-приложение, с использованием любых технологий стека C#. 
- Приложение должно включать в себя:
- Страницу с гридом (ExtJS любой версии, на крайний случай можно использовать другой знакомый JS фреймворк). Грид отображает записи из SQL таблицы (ФИО, Дата рождения, Телефон)
Требования:
- Данные в грид должны загружаться с использованием Ajax запросов.
- Должна быть реализована возможность добавления/редактирования значений прямо из грида.
- После редактирования – грид обновляется тем-же Ajax запросом.
